#### masraff assignment - backend  
  
---  
  
##### Framework:  
 * NestJS (NodeJS Framework)
 * Swagger
 * JWT authentication
 * PostgreSQL

```

├── boilerplate.polyfill.ts
├── common

├── exceptions

├── guards

│   ├── public
│   ├── user
│   └── vat-rate
├── providers
├── shared

└── viveo-swagger.ts
```
  
##### Installation:  
```
 git clone https://github.com/avnigenc/masraff-backend
 cd masraff-backend 
 npm install 
 ng serve  
```
